# Multi-Language Support (i18n)

## Summary

Add multi-language support to Reading Circle with English and Dutch. Uses react-i18next for the frontend, per-user locale stored server-side, and translated email templates.

## Backend

### Database
- Add `locale` TEXT column to `users` table, default `'en'`

### API
- `PATCH /api/auth/locale` — update authenticated user's locale (`{ locale: 'nl' }`)
- Include `locale` in `GET /api/auth/me` response and login response
- Add `locale` to `UserResponse` shared type

### Emails
- All email functions accept a `locale` parameter
- When sending notifications, look up each recipient's locale
- Translate email subject lines and body text

## Frontend

### Dependencies
- `react-i18next` and `i18next` (bundled, no HTTP loading)

### File Structure
```
client/src/i18n/
  en.json     — English translations
  nl.json     — Dutch translations
  index.ts    — i18n initialization
```

### Translation Keys
Flat namespace with page-based prefixes:
- `nav.*` — navigation labels
- `common.*` — shared buttons/labels (Save, Cancel, Delete, etc.)
- `books.*` — books page
- `meets.*` — meets page and detail
- `dashboard.*` — dashboard
- `admin.*` — admin page
- `settings.*` — settings page
- `auth.*` — login, join, setup, forgot/reset password
- `members.*` — members page and profiles

### Language Switching
- Settings page: language dropdown, calls `PATCH /api/auth/locale` then switches i18next
- Pre-auth pages: small language toggle (stored in localStorage until login)
- After login: load user's locale from `/api/auth/me` response

### Initialization
- i18n initialized in `main.tsx` before React render
- Default language: `'en'`
- After auth loads: switch to user's stored locale

## Scope

### Pages to translate (all 14):
- LoginPage, SetupPage, JoinPage, ForgotPasswordPage, ResetPasswordPage
- DashboardPage, BooksPage, BookDetailPage
- MeetsPage, MeetDetailPage
- MembersPage, MemberProfilePage
- AdminPage, SettingsPage

### Other:
- Layout.tsx (navigation)
- Email templates (invitation, password reset, voting opened, book selected)
